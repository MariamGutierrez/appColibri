Bienvenido a la API de PQRS

Endpoints disponibles:

  • /usuarios - Obtener todos los usuarios (GET)
  • /pqrs - Obtener todas las PQRS (GET)
  • /login - Login (GET)
  • /re - Login (GET)
  • /pqrs - Crear una nueva PQRS (POST)

Ejemplo de uso:

{# (esto es para hacer lo que haces en postman en la terminal de linux, pero si no solo ponlo normal en el postman) #}
curl -X GET http://localhost:5000/usuarios
curl -X GET http://localhost:5000/pqrs
curl -X POST http://localhost:5000/pqrs -H "Content-Type: application/json" -d '{"tipo": "queja", "descripcion": "Problema con el servicio", "estado": "pendiente", "usuario_id": 1}'
